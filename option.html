G-Tags | Google Analytics tags tracker By Schneider Electric

Works with Chrome v44 and further.

How to use :

  • Click on grayed icon: G-Tags window is displayed, icon is highlighted, tags recording begins.
  • Click on highlighted icon to display G-Tags when overlapped or minimized.
  • Uncheck Pages or Events or Others to hide them (they are still recorded, so they will be displayed when checked again).
  • Check Page URL to see URL that triggered Page tag.
  • Check Details and Advanced to see more data for all tags, or select a tag and double click.
  • Click on orange led to stop/resume recording.
  • Click on chevron to collapse control bar (or double click anywhere on it), click on collapsed bar to expand it.
  • Double-click on tag type to show/hide all its details.
  • Double-click on tag values to copy them.
  • Mouse hover advanced details to see parameters meaning.
  • Click on copy icon to copy all displayed entries in a tabbed format that can be pasted in a text editor or as columns in Excel.
  • Click on trash icon to remove all recorded tags.
  • Close X-Tags window: icon is grayed, tags recording ends.

Privacy :

  • G-Tags doesn't transmit any of your private data and will never do it.

How to uninstall :

  1. Right click on G-Tags icon.
  2. Select Remove from Chrome.
  3. Click on Remove button.
